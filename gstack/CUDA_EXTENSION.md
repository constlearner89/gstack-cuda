# CUDA / Simulation Extension

This repository now includes a simulation-oriented extension layer for CUDA/C++ numerical software.

## Included Skills

- `/cuda-office-hours`
- `/cuda-plan-eng-review`
- `/cuda-dev`
- `/cuda-investigate`
- `/cuda-review`
- `/cuda-benchmark`
- `/cuda-verify`
- `/cuda-release`

## Intended Workloads

- DEM
- SPH
- FVM
- FEM
- CFD
- multiphysics solvers
- GPU kernel optimization
- verification and benchmark discipline for numerical codes

## Usage Model

Keep the original gstack skills for general planning and safety, but prefer the CUDA skills when the task is about solver architecture, discretization, numerical correctness, GPU debugging, or kernel performance.
